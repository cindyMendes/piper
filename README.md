# Piper 🐾 

Carte d'identité web pour Piper, au cas où elle se perd.  
Accessible via un **QR code** imprimé sur son collier.

## 📁 Structure

```
piper/
├── index.html   → page principale
├── style.css    → styles
└── README.md
```

## 🚀 Déploiement (Netlify)

1. Push le repo sur GitHub
2. Connecte le repo sur [netlify.com](https://netlify.com)
3. Netlify génère une URL publique (ex: `piper.netlify.app`)
4. Génère un QR code avec cette URL sur [qrcode-monkey.com](https://qrcode-monkey.com)
5. Imprime et colle le QR code sur le collier de Piper 🐾

## ✏️ Modifier les infos

Toutes les informations sont dans `index.html`, directement éditables.
